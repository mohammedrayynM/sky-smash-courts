# Sky Smash Courts

A premium sports booking platform for Turf Football, Badminton, and other activities.

## Tech Stack

- **Frontend**: Next.js 14, Tailwind CSS
- **Backend**: Node.js, Express, MongoDB
- **Authentication**: JWT

## Getting Started

### Prerequisites

- Node.js
- MongoDB

### Installation

1. Clone the repository.
2. Install dependencies for server:
   ```bash
   cd server
   npm install
   ```
3. Install dependencies for client:
   ```bash
   cd client
   npm install
   ```

### Running the App

1. Start the server:
   ```bash
   cd server
   npm run start
   ```
2. Start the client:
   ```bash
   cd client
   npm run dev
   ```
